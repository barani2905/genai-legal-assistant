# GenAI Legal Assistant (MVP)

## Problem Statement
Small and Medium Enterprises (SMEs) often struggle to understand legal contracts due to complex language and high consultation costs.

## Solution
This project is a GenAI-powered Legal Assistant that helps users:
- Upload legal contract text
- Understand key clauses in simple language
- Get AI-generated summaries and explanations

## Tech Stack
- Python
- Streamlit
- Large Language Model (LLM)

## How to Run Locally
1. Install dependencies
2. Run: streamlit run app.py
3. Open browser at http://localhost:8501

## Outcome
The application simplifies legal documents and helps SMEs make informed decisions quickly and affordably.
